## Zadatak 📝

1. HTML dokument (index.html) treba promijeniti pomoću JS-a i HTML DOM API-ja tako da:
	- Se promijeni pozadinska boja dokumenta svaki put kad se promijeni temperatura

- Temperatura treba biti nasumično odabrana po sljedećoj formuli:
	- Nasumični broj * 80 - 35

- Nasumičnu temperaturu treba ubaciti u element div s klasom temperature prilikom prvog učitavanja stranice te prilikom svakog klika na gumb "Get temperature"

- Uvjeti za prikaz temperature:
	- Temperatura je viša od 40°C - pozadinska boja dokumenta je #FF4D00
	- Temperatura je viša od 10°C - pozadinska boja dokumenta je #FFBD00
	- Temperatura je viša od -10°C - pozadinska boja dokumenta je #01DFFF
	- Ostali uvjeti - pozadinska boja dokumenta je #0045FF
